About

This service was created by Jonathan Birge and Alex Birge as an educational father-son project. It is in no way affiliated with Tesla, Inc.

If there turns out to be any interest in this, future versions may include features such as links based on your current location, customizable news sources, and persistent user customization of the interface.

For questions or feedback, please email admin@teslas.cloud or submit an issue to the GitHub repository. Even a quick note letting us know you’re using this would be motivation to keep improving it.

Apologies to those outside of the US; right now some functionality is limited to CONUS. If you’re using this from another region, please let us know where you're using this (via the contact email above) and we'll see what we can do to improve support for your region with news and weather maps.

Tips

  • Bookmark this page using the star in the browser menu.
  • You can make the browser easily available by dragging it to the left-hand side of the dock at the bottom of the screen.
  • Dark mode is turned on automatically based on sunset at your location. However, you can also toggle it manually via the button in the upper-right.
  • You will find that some links will open inside the frame for convenience. To return to the list of links, just click on one of the sections on the left.
