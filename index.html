About

This service was created by Jonathan Birge as a personal project. It is not affiliated with Tesla, Inc.

If there turns out to be any interest in this, future versions may include things like better inline weather information and maps, local links based on your current location, and per user customization.

Apologies to those outside of the US; right now some functionality is limited to CONUS. If you're using this from another region, please let me know (see below) where you are and, if possible, recommend a source for weather maps and data for your region.

For questions or feedback, please email admin@teslas.cloud or submit an issue to the GitHub repository. Even a quick note letting me know you're using this would be motivation to keep improving it!

Tips

Bookmark this page using the star in the browser menu. You can make the browser always easily available by dragging it to the left-hand side of the dock at the bottom of the screen.

Dark mode is currently set automatically based on sunset at your current location. If you toggle it manually (via the button in the upper right) it will disable automatic dark mode.

Some links will open inside this frame for convenience (not all websites support this). To return to the list of links, just click on one of the sections on the left.

Bugs

Too many to count right now! This is very much a project in progress. The biggest known bug right now is the incorrect time zone in the sunset and sunrise times. Apparently the Tesla browser time uses the time zone of the proxy server (which is in California for US users) not the actual GPS location of the car. Definitely a fixable problem but I haven't gotten around to it yet.
