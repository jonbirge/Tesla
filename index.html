About

This service was created by Jonathan Birge as a personal project. It is not affiliated with Tesla, Inc.

If there turns out to be any interest in this, future versions may include things like better inline weather information and maps, local links based on your current location, and per user customization.

Apologies to those outside of the US; right now some functionality is limited to CONUS.

For questions or feedback, please email admin@teslas.cloud or submit an issue to the GitHub repository.

Tips

Bookmark this page using the star in the browser menu. You can make the browser always easily available by dragging it to the left-hand side of the dock at the bottom of the screen.

Dark mode is currently set automatically based on sunset at your current location. If you toggle it manually (via the button in the upper right) it will disable automatic dark mode.
